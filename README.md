# 🗳️ Online Voting System

A secure **web-based voting platform** developed using **PHP, MySQL, HTML, CSS, and JavaScript**.  
The system allows users to register, login, cast their votes, and view results while ensuring that each voter can vote only once.

This project demonstrates **full-stack web development**, database integration, and session-based authentication.

---

## 🚀 Features

- User Registration System
- Secure Login Authentication
- Cast Vote for Candidates
- Prevent Multiple Voting
- Admin Dashboard
- Real-time Vote Counting
- Result Display System
- MySQL Database Integration

## ⚙️ Installation Guide

### 1️⃣ Clone the repository
git clone https://github.com/chethankumarsv/online-voting-system.git

### 2️⃣ Move project folder

Move the project folder to:
xampp/htdocs/

### 3️⃣ Start server

Open **XAMPP Control Panel** and start:

- Apache
- MySQL

### 4️⃣ Import Database

1. Open **phpMyAdmin**
2. Create a database named:
voting_system

3. Import the SQL file.

### 5️⃣ Run the project

Open browser and visit:
http://localhost/online-voting-system
